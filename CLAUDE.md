# Claude Code Workflow Lab

## Requirements

- Git
- Node.js >= 20
- pnpm
- An editor with Claude Code enabled

## Commands

### Install / Setup

    pnpm install

### Development

    pnpm test
    pnpm run lint
    pnpm run build

### Other useful commands

    pnpm run dev

> Claude: use the commands above when you need to install, test, or run this project.
